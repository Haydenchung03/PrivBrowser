# Privacy Focus Browser

This is a simple desktop browser made with PyQt6.

It’s built to help you stay a bit more aware of how you’re using the internet, without getting in your way.

---

## What it does

- Lets you browse normal websites
- No ads or tracking added by the browser
- No AI features
- No data collection
- You pick what you're doing (like Work, Study, etc.)
- You set a time limit for your session
- It shows a small timer while you browse
- When time runs out, you choose:
  - Start a new task
  - Close the browser
